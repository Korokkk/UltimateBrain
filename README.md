This is a quiz platform where users can ...

register, login and logout

Make a new quiz including several questions (quiz admin)
	for each question, user can specify its question content, choices, answer time, how many marks this question worth, upload image and video for this question, and of course the correct answer.
	this quiz allows multiple player to answer simultaneously.

Join a quiz made by other users (player)
	wait for quiz maker to begin, after that, 
	answer each question and when this question's time is out, then the answer of it will be displayed, then wait for admin to go next question, player choose answer,  time out, show answer, admin go next, until all questions are finished
	show this player's result of the quiz, also top 5 player will be displayed.
